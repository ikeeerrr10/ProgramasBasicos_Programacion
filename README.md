# Programas Básicos para programación

## :pushpin: Descripción

Este repositorio contiene pequeños programas realizados en **Java**, pensados para practicar y afianzar los conceptos básicos del lenguaje.
Son ejercicios sencillos, ideales para quienes estén comenzando a programar.

## :rocket: Objetivo del repositorio

- Aprender la sintaxis básica de Java
- Practicar estructuras como condicionales y bucles
- Entender la lógica detrás de programas simples

## :open_book: Contenido

El repositorio incluye ejemplos como:

- Programas de salida por pantalla
- Uso de variables y tipo de datos
- Estructuras de control básicas

## :bookmark: IDE utilizado y donde descargarlo

El IDE utilizado para el desarrollo de los programas ha sido **Intellij IDEA**
